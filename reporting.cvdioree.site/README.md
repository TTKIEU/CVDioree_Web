## AI Usage
Throughout the entirety of this project, AI was used in several ways. For starters, we used it to help us build the general structure and template of many of the features. With a template being built, it would allow our group to easily expand on it as not only much of the messy and possibly easily confusing work would already be completed. Another important way to note how AI was used was to help us ensure that our code followed the MVC architectural structure. As it has been a while since we familiarized with this strucure on a project level, using AI to ensure that it followed and met the MVC structure has allowed our work flow to be clean and consistent. A final way how our group used AI was to simply aid us in tasks we simply are not strong in, such as using SQL to help manage logging, and users. \
\
The way how AI was used in this project demonstrated that it can be a valueable tool. However, the value of such a tool is only demosntrated when you understand the concepts of what you are implementing, and how it should be implemented. Having this understanding is important as it allows you clearly understand what the AI is typing, which then also allows you the ability to know how to modify, add, or remove what the AI did so that it better fits your project, since it is only the individual who can envision what the final product should look like.
